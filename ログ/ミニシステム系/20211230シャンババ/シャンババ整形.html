１．はじめに

　これはファンタジー系のTRPGのルールブックなのを教える
　ゲームバランスは…（意味深に無言）

　シナリオの作り方、サンプルシナリオは1番下にあるのを教える


２．大雑把な説明
　


　魔法があってモンスターがいて王国があって冒険者ギルドがあって…みたいな世界
　そんな感じのファンタジーものっぽい世界で冒険してもらうことを教える

　PCは冒険者になってモンスターをぶっ殺したり、探し物を探しに行ったりしてもらう
　冒険者はダンジョンに潜ったり…クエストを受けてモンスターを倒しに行ったり…
　そういうことをする仕事であることを教える　
　
　PCの目標はとりあえず死なないことなのを教える
　クエストクリアが出来なさそうなら諦めておうちに帰ったりしてもいいのを教える
　
　勝利も敗北も知り、逃げ回って涙を流して男は一人前になる
　泣いたっていいんだ・・・!!　乗り越えろ!!!
　



３．キャラクター作成

　キャラクターを作成するぞォ～～～！！！

・能力値

　DEX　　…　（敏捷性:行動順や回避などに使う）　3D6

　POW　　…　（精神力:魔法を使う時に使う）　　　3D6

　INT　　…　（知性:頭の良さ、器用さの時に使う）3D6

　
x5 3D6 とか 15D6 みたいにまとめて振ると楽なのを教える



・ステータス

　HP　 　…　 (CONの2倍)＋10（0になると戦闘不能）

MP 　…　 POW＋5

　行動順 …　 DEX

・固有技能

　固有技能はそのキャラクターがジョブとか関係なく持ってる特殊な力のことだと教える　シャンクトゥルフの捏造技能といっしょ

　なんでもいいから2つ決めてくれ(^^)
　固有技能はスキルとして使えることを教える

　●（固有技能の名前）：消費MP5　使う時に下から１つ効果を選ぶ
　・単体に5D6のダメージの攻撃を行う
　・全体に4D6のダメージの攻撃を行う
　・自分の判定の直前に使う。判定のダイスを1つ増やす


・持ち物

　最後は持ち物だァ～～～～！
　剣や斧、ナイフ、魔法の杖、ピッキングツール…
　とにかく持ってそうなものを持たせればいいことを教える



４．スキル


　ちょっと待てスキルってなんだよ…
　スキルはMPを消費して使える特殊な技みたいなもんなのを教える
　MPがなくなると使えないから要注意なのを教える




４．ダイスロール


　判定は基本的に　　2D6+（使用する能力値）＝達成値　なのを教える

　GMが難易度を教える→判定する→達成値≧難易度→成功
　　　　　　　　　　　　　　　→達成値＜難易度→失敗

　ダイスの出目が６が2つ以上出たらクリティカルでいいことが起こるのを教える
　ダイスの出目が全部１が出たらファンブルで悪いことが起こるのを教える
　基本的には達成値に関わらず成功にしたり失敗にしたらいいことを教える

　
・対抗ロール

　攻撃に対する回避、逃げる相手を追いかける、口論など…勝敗を決めたい時が起こった
　両者が判定して達成値が高い方の行動が成功することを教える
　これを対抗ロールと呼ぶことを教える

　同値だったらお互い1D6振って高い方が勝ちなのを教える
　それも同値だったら…（意味深に無言）

◇主な判定に使う能力値

　・通常攻撃・力仕事　　　　…　STR
　・受け流し・免疫力　　　　…　CON
　・通常攻撃・回避　　　　　…　DEX
　・調査・アイデア・魔法　　…　INT
　・魔法を使う・恐怖に勝つ　…　POW
　・固有技能　　　　　　　　…　好きな能力値

５．戦闘ルール


　本作での戦闘は、（攻撃）→命中→回避or受け流し→ダメージという一連のダイスを振っていくことによって行うことを教える
　また行動順が回ってきても必ずしも攻撃をしなければいけないわけでは無く、仲間の回復をしたり次の攻撃の準備をしたりできる
　さらに足元を狙うなど細かい指定もRPとして可能なことを教える
　GMを説得できれば補正が貰えるぞォ～！
　あとスキルを使うときは使う前に宣言してくれ(^^)

・行動順

　その戦闘シーンにいるＰＣと敵の中からＤＥＸが高い順に行動ができることを教える
　ただし発狂していたり拘束されているとそれが解けるまで動けない　南無…

　全員が行動し終えるまでを１ターンとして数える


・攻撃

　まずはどのような攻撃をするのか宣言しよう
　武器や素手で攻撃するのが”通常攻撃”、魔法を使った攻撃が”魔法攻撃”
　●スラッシュ　や●マジックブラスト　みたいなスキルを使った攻撃でもいいことを教える
　
　どのように攻撃をするか決めたら次は命中の判定を行う
　通常攻撃ならSTRかDEX、魔法攻撃ならPOWかINTで判定することを教える
　●（固有技能の名前）で攻撃するなら好きな技能でいいのを教える

　攻撃することもあればされることもある
　その時は回避するか受け流すか諦めるか選べることを教える

　回避　　　…　DEXで判定。成功すればダメージ０
　受け流し　…　CONで判定。成功すればダメージ半減
　諦める　　…　そのまま攻撃を喰らう

・ダメージ

　攻撃が当たったらダメージ算出だァ～～！！
　ダメージは通常攻撃は　1D6+STR、魔法攻撃は　1D6+POW　なのを教える

・HPが0になった場合

　HPが0になると戦闘不能になることを教える
　戦闘不能のまま放置すると死ぬ　悲しいだろ
　死ぬまでに回復させれば死なずに済むことを教える

１.ハウンマヌヘッ

このゲームは原始時代の人類の生活を体験しようという社会教材TRPGなことを教える

２.ブルヘウンハ

以下の8つから2つまで選べることを教える

・言語　持ってると言語を話せる
・狩猟　獲物へのダメージが２ダメージになる
・索敵　持ってると索敵時にＧＭと同じ出目として扱える
・頑強　ＨＰが５になる
・護衛　攻撃を受けた味方をかばうことができる
・治療　自分のターンで味方１人のＨＰを全回復できる
・調理　獲物を倒した際に獲得できる肉の数が１増える
・家族　ＨＰが０になったとき１度だけＨＰを最大まで回復できる

３.ホフンマ　ハッハッ

ゲームの流れとしては狩りに出かけ、索敵し、獲物を倒して肉を獲得し、持ち帰って宴という感じなことを教える
肉の目標数はＰＣの人数×３で、これを達成するとゲームクリアなことを教える

・索敵　ＧＭはＰＣの人数分、ＰＬは１人１回ずつ６面ダイスを振る
　　　　ＧＭとＰＬで出目が同じなダイスが多いほど、よい獲物を見つけられることを教える

・戦闘　それぞれ６面ダイスを振り低い順から攻撃できる
　　　　ＰＣはブルヘウンハが無い限り、ＨＰは３、攻撃は１なことを教える
　　　　原始時代なので命中や回避判定はない
